> A modern, minimal, and premium personal portfolio website built with pure HTML, CSS, and JavaScript.

## 🎯 Overview

This is a personal portfolio website for **Umar Maulana** — a Full Stack Web Developer and Photographer. The website showcases professional work in web development and photography, building personal branding, and attracting potential clients.

## ✨ Features

### Design & UX
- 🎨 Modern, minimal, and premium design
- 📱 Fully responsive (Desktop → Mobile)
- 🖼️ Large typography with clean hierarchy
- 🔲 Grid-based layout with ample white space
- 🌊 Smooth scroll and elegant animations
- ✨ Hover effects with orange accent

### Sections
- 🏠 **Hero** - Fullscreen with typing effect
- 👤 **About** - Personal journey and vision
- 💼 **Services** - Web Development & Photography
- 📁 **Portfolio** - Filterable project gallery
- 📸 **Photography** - Masonry gallery with lightbox
- 🛠️ **Skills** - Progress bars with animations
- 📅 **Experience** - Professional timeline
- 💬 **Testimonials** - Slider with client feedback
- 📬 **Contact** - Form with social links

### Technical Features
- 🚀 Google PageSpeed 95+ (Desktop) / 90+ (Mobile)
- 🖼️ Lazy loading images
- ⚡ Minified CSS & JavaScript
- 🔍 SEO optimized with meta tags, Open Graph, Schema
- ♿ Accessibility focused (ARIA, keyboard navigation)
- 📱 Touch-friendly on mobile devices

## 🛠️ Technologies Used

### Core
- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Flexbox, Grid
- **JavaScript ES6+** - Vanilla JS, Intersection Observer

### Fonts
- [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) - Headings
- [Inter](https://fonts.google.com/specimen/Inter) - Body text

### Icons
- Custom SVG icons (no external icon libraries)
